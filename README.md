# Bili_YKChengZi 项目站点

## 项目介绍

本仓库是 Bili_YKChengZi 的官方网站源码，对应线上站点为 [chengzi.funnyr.xyz](http://chengzi.funnyr.xyz)。网站主要展示 Bili_YKChengZi 相关产品及信息，目前处于产品仍在制作中，敬请期待。

## 网站功能

- 展示 Bili_YKChengZi 的产品。
- 提供下载链接。
- 提供产品简介。

## 技术栈

- HTML5 作为页面结构基础
- CSS3 结合 Bootstrap 进行样式设计
- JavaScript 实现交互效果
- Font Awesome 提供图标支持
- GitHub Pages 部署展示

## 页面结构

- 相关样式文件位于 `css/` 目录
- 字体图标位于 `fontawesome/` 目录
- 图片资源位于 `img/` 目录
- 脚本文件位于 `js/` 目录
- 视频资源位于 `video/` 目录

## 部署说明

项目通过 GitHub Actions 自动部署到 GitHub Pages，部署配置文件为 `.github/workflows/deploy.yml`，每次推送到 `main` 分支会自动触发部署流程。
